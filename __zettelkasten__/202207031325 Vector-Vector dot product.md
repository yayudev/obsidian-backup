# Vector-Vector dot product
----
- A dot product (x . y) is the sum of the multiplying of each correspongin element.
- Only vectors the same size are compatible for dot product.
- Example:
```python
[1, 2] . [10, 20]
= (1*10) + (2*20)
=   10   +   40
=       50
```
