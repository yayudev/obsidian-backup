# Matrix Transposition
----
- Consists on exchanging the rows and columns in a matrix.
- Is the most common form of reshape.
- Example:
```python
# With the matrix
[
  [0, 1, 2],
  [3, 4, 5],
  [6, 7, 8]
]

# Applying a transpoisition it will result in
[
  [0, 3, 6],
  [1, 4, 7],
  [2, 5, 8]
]
```
